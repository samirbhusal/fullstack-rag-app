# Vite Template

## Getting Started

## To run frontend

1. Run `npm install`
2. Run `npm run dev`

## To start backend:

1. cd backend / go to backend directory
2. enter `uvicorn app:app --reload` command
3. Or Option 2: Specify the Path: `uvicorn src.backend.app:app --reload` or `./.venv/bin/python -m uvicorn app:app --reload`
